How it works

1
VADER scoring

The raw text is scored using VADER (Valence Aware Dictionary and sEntiment Reasoner), a lexicon and rule-based model tuned for social and expressive language.

2
NLP preprocessing

Separately, the text is tokenised, stop-words removed, and words lemmatised to extract the most meaningful content keywords.

3
Persisted history

Every analysis is stored in a database so you can track sentiment trends over time across multiple submissions.
